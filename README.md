# Saurabh Kumar Dubey — Portfolio

A personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**, showcasing skills, projects, experience, and contact information.

## Tech Stack

- **React 19** — Component-based UI
- **Vite 6** — Fast build tool & dev server
- **Tailwind CSS 4** — Utility-first styling
- **Framer Motion 12** — Animations & transitions
- **React Router 7** — Client-side routing
- **React Icons** — Icon library

## Sections

- **Hero** — Introduction with role & tagline
- **About** — Brief personal summary
- **Skills** — Technical proficiencies (HTML, CSS, JS, React, Node.js, MongoDB, etc.)
- **Experience** — Education & internship timeline
- **Projects** — Featured project cards (E-Commerce Platform, Weather Dashboard, Task Management App)
- **Contact** — Get in touch form / links

## Getting Started

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```
